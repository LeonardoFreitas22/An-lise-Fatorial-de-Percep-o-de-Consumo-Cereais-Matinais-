# Análise Fatorial de Percepção de Consumo Cereais Matinais
Análise Fatorial aplicada a cereais (dados Kellogg) para reduzir 25 variáveis de percepção em 4 pilares: Saúde, Artificialidade, Tradição e Indulgência. Utilizou Python, testes KMO/Bartlett e rotação Varimax. O projeto converte dados qualitativos em insights para posicionamento de marca, otimização de marketing e inovação de produtos.
